**💡 刷题建议：**
1. **先横后纵**：先掌握“基础技巧”和“常用数据结构”，再攻克“动态规划”和“图论”。
2. **注重总结**：每刷完一类题目，尝试总结该类题目的通用模板。
3. **软实力同步**：在刷题时，有意识地练习“思路清晰”地书写代码，模拟面试表达。
